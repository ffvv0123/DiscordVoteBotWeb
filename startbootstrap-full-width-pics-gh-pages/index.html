코로나방 방장선거 규칙

1. 기본 룰

- 4시간마다 티켓을 한 장 받을 수 있습니다.

- 서버가 터지면 시간에 맞춰서 추가지급 해드립니다.

- 티켓의 갯수는 음수가 될 수 없습니다.

- 추가예정

투표봇 명령어

1. 명령어 기본형식

- 모든 명령어는 !를 붙이고 시작합니다.

- 후술할 명령어에서 [사용자], [숫자]가 표시되어 있다면, 사용자와 숫자를 사용한다는 뜻입니다.

- 명령어의 키워드는 공백으로 구분됩니다.

- 상태를 설정하는 명령어에서는 기본적으로 공백을 지원하지 않습니다.

- 다만, 큰따옴표("")로 문자열을 감싸면 공백을 넣을 수 있습니다.

- 숫자는 양수, 음수, 0 모두 지원합니다. 다만, 15자리가 넘어가는 수는 오류가 발생할 수 있습니다.



2. 시스템 명령어(관리자 전용)

!부정선거 : 투표봇 데이터베이스를 초기화 합니다.

!로비 [사용자] [숫자] : [사용자(로비 대상)]에게 [숫자]개의 티켓을 부여합니다.

!압수수색 [사용자] [숫자] : [사용자(압수수색 대상)]에게 [숫자]개의 티켓을 수거합니다.

!대리출마 [사용자] : [사용자(출마 대상)]를 직접 데이터베이스에 등록합니다.



3. 개인 명령어

!도움말 : 디스코드봇이 도움말이 적힌 임베드를 출력합니다. 여기에는 선거 규칙, 투표봇 사용법이 포함되어 있습니다.

!대선출마 : 본인을 데이터베이스에 등록합니다. 모든 값들은 기본값으로 설정되어 있습니다.

!녹봉 : 사용자에게 티켓 한 장을 줍니다. 4시간이 지나면 다시 사용할 수 있습니다.

!기호 [설정할 기호] : 사용자의 기호 번호를 설정합니다. 원래는 숫자만 가능해야 하지만 이상하게도 문자까지 됩니다.

!정당 [설정할 정당 이름] : 사용자의 정당을 설정합니다. 개발자는 본인의 정당이름에 대해 어떠한 책임도 지지 않습니다.
간첩신고: 111

!자기소개 [설정할 자기소개] : 사용자의 자기소개를 등록합니다.

!내정보 : 본인의 정보를 조회합니다.



4. 상호작용 명령어

!투표 [사용자] [숫자] : [사용자(지지하는 사람)]에게 [숫자]만큼의 표를 보냅니다.

!출마현황 : 모든 후보들의 정보를 확인할 수 있습니다.

!사상검증 [사용자] : [사용자(조회대상)]이 누구에게 투표했는지 볼 수 있습니다.
